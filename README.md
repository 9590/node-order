> A node-based order interface for E-Business


## MindMap

![Map](/assets/order.png)

## Install

```sh
$ npm install --save node-order
```


## Usage

```js
var nodeOrder = require('node-order');

nodeOrder('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
